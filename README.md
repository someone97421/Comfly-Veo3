# Veo3 视频生成 App (Veo3 Video Generation App)

## 中文说明

### 项目简介

这是一个基于 Web 技术的视频生成应用前端界面，专为 Veo3 模型设计。该应用提供了一个现代化、响应式的用户界面，允许用户通过提示词（Prompt）和多图参考来生成高质量视频。

### 核心特性

* **双色主题切换**：支持深色（默认，科技蓝）和浅色（清新绿）两种视觉模式，适应不同使用场景。
* **多图参考系统**：提供多个图像插槽，支持拖拽上传、点击上传以及直接粘贴图片（Ctrl+V），方便用户提供视觉参考。
* **毛玻璃视觉设计**：采用现代化的 Glassmorphism 设计风格，界面通透且富有层次感。
* **响应式布局**：基于 Tailwind CSS 构建，完美适配不同尺寸的屏幕。
* **实时交互反馈**：内置加载动画、平滑滚动及消息提示系统，提升用户操作体验。

### 技术栈

* **HTML5 & CSS3**：基础结构与自定义样式。
* **Tailwind CSS**：用于快速构建响应式布局。
* **JavaScript (ES6+)**：处理图像上传逻辑、主题切换及 API 交互。
* **Google Fonts (Inter)**：提供清晰、现代的排版字体。

## English Description

### Project Overview

This is a web-based front-end interface for a video generation application, specifically designed for the Veo3 model. It features a modern, responsive UI that enables users to generate high-quality videos using prompts and multiple reference images.

### Key Features

* **Dual Theme Support**: Includes Dark mode (default, tech blue) and Light mode (fresh green) to suit different environments.
* **Multi-Image Reference System**: Multiple image slots supporting drag-and-drop, click-to-upload, and direct clipboard pasting (Ctrl+V).
* **Glassmorphism UI**: Uses a modern glass-like aesthetic for a sleek and layered interface.
* **Responsive Design**: Built with Tailwind CSS to ensure a seamless experience across all device sizes.
* **Real-time Feedback**: Features built-in loading animations, smooth scrolling, and an interactive message system for enhanced UX.

### Tech Stack

* **HTML5 & CSS3**: Core structure and custom styling.
* **Tailwind CSS**: Rapidly built utility-first responsive layout.
* **JavaScript (ES6+)**: Handles image upload logic, theme switching, and API interactions.
* **Google Fonts (Inter)**: Provides clean and modern typography.
